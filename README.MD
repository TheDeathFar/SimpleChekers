# Консольные шашки.

---

Простая игра в шашки, которая полностью написанна на Java. Доска и фигуры отображается в консоли как Unicode-символы.

Реализованы основные классы:

1) Figure
2) Bot и Player
3) Cell
4) Color
5) Direction   
6) Move
7) Game
8) Application

И ещё несколько вспомогательных.
